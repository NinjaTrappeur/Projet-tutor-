Présentation

Ce projet regroupe les classes utilisées concomitamment par les deux applications, Jade et Fractal.

Description des packages

  1. message: contient les interfaces et les imlémentations des différents types de messages échangés dans l'application mixte.
  2. remote: contient l'interface de tout fournisseur de service (à priori distant) de type agence de voyage.
  3. shared.agent: contient l'agent qui est destiné à être utilisé comme composite dans un composant Julia.
  4. shared.behaviour: contient le behaviour (automate) de l'agent décrit ci-dessus.
  5. shared.utils: contient un lanceur de conteneur, permettant de lancer l'agent décrit ci-dessus dans un programme.
  6. webservice.message: contient la traduction, sous forme de JavaBeans, des classes du package message.
