Présentation

Ce projet regroupe les classes utilisées concomitamment par les deux application, Jade et Fractal.

Description des packages

  • message: contient les interfaces et les imlémentations des différents types de messages échangés dans l'application mixte.
  • remote: contient l'interface de tout fournisseur de service (à priori distant) de type agence de voyage.
  • shared.agent: contient l'agent qui est destiné à être utilisé comme composite dans un composant Julia.
  • shared.behaviour: contient le behaviour (automate) de l'agent décrit ci-dessus.
  • shared.utils: contient un lanceur de conteneur, permettant de lancer l'agent décrit ci-dessus dans un programme.
  • webservice.message: contient la traduction, sous forme de JavaBeans, des classes du package message.
